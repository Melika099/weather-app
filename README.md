> SkyVision — Weather App

A real-time weather app that fetches live data for any city in the world.

>Live Demo
[puseletso-weather-app.netlify.app](https://puseletso-weather-app.netlify.app)

> Features
- Search any city in the world
- Shows temperature, humidity, wind speed, visibility and UV index
- Quick search pills for popular cities
- Press Enter to search
- Fully responsive design
